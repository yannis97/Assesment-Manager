# Assesment-Manager
 This app is developt with Java. The purpose of this app is to help teachers to encode their students assesments and make statistics with those.

Installation

* Télécharger le dossier du projet sur github ou le zip contrenant le projet

* Extraire le zip du projet

* Dans le dossier « Assesment-Manager » , récupérer le dossier « Assesment Manager Project » et le placer dans votre workspace eclipse

Exécution de l’Application

* Ouvrir le fichier mainclass

* Faire un clic droit puis cliquer sur « Run as » 

* Sélectionner « 1 Java Application »

Exécution des test unitaires

* Ouvrir le fichier testmainclass

* Faire un clic droit puis cliquer sur « Run as »

* Sélectionner « 1 Junit Test »

NB : Dans le cas où la  1ère ligne d’importation n’est pas reconnu, cela signifie que JUnit5 n’a pas été sélectionné ou n’est pas dans le class path, dans ce cas : 

* Faire un clic droit sur le projet puis « Properties »

* Cliquer sur « Java Build Path » puis « classpath » , puis « Librairies »

* Cliquer sur « Add Librairies.. » puis sélectionner JUnit -> JUnit5

* Cliquer sur « Finish » puis « Apply »

* Assurez vous que JUnit5 est bien sélectionné pour la testmainclass en effectuant un clic droit dans le fichier de la classe en question , puis « Run As » , « Run Configuration » .

* Sélectionner « JUnit » , puis « JUnit testmainclass », mettre JUnit5 en TestRunner

* Vous pouvez dés à présent Run les différents tests.
